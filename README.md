# TGT Весенняя технологическая практика (Frontend)
## Чтобы запустить проект
1. Запуск проекта
    * Собрать и запустить проект
      ```commandline
      docker-compose up -d --build
      ```
    * Запустить проект
      ```commandline
      docker-compose up -d
      ```
    * Остановить проект
      ```commandline
      docker-compose stop
      ```
